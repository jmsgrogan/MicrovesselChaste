Microvessel Chaste

An add-on for modelling microvessels with Chaste.

Download this project as a .zip file Download this project as a tar.gz file

Features

  • 2D and 3D on- and off-lattice agent based models of cells and microvessels
  • Integration with Chaste
  • Blood flow with haematocrit partitioning, structural adaptation and regression
  • Sprouting angiogenesis
  • Nutrient transport PDEs solved using finite difference or finite element methods
